🚀 IdeaVault – Share, Explore & Grow Innovative Ideas

🌟 Project Overview

IdeaVault is a modern full-stack idea sharing platform built with Next.js where users can create, explore, and interact with innovative ideas from the community.
The platform includes secure authentication, personalized profiles, idea management, comments, and interactive user features — all designed with a clean and responsive UI.

This project demonstrates modern web development practices using Next.js App Router, Better Auth, and scalable component-based architecture.

🔥 Live Features

✅ User Authentication & Authorization using Better Auth
✅ Secure Login & Registration System
✅ Create, Update & Manage Ideas
✅ Comment & Interaction System
✅ Personalized User Profile Section
✅ Trending Ideas Showcase
✅ Responsive Modern UI Design
✅ Reusable Components Architecture
✅ Dynamic Navigation & Footer
✅ Protected Routes & Session Handling

✅ Dark/Light Friendly Design
✅ Fully Built with Next.js App Router
